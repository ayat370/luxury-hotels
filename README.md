## Luxury Hotels
Application web de gestion d’hôtels et de réservations.

### Technologies
- PHP
- MySQL
- HTML
- CSS

⚠️ Ce projet nécessite un serveur PHP/MySQL (XAMPP, WAMP ou hébergement web).
GitHub sert ici à la gestion du code source.

### Installation
1. Cloner le repository
2. Importer le fichier `database.sql` dans votre base MySQL
3. Modifier `config.php` avec vos identifiants locaux
4. Lancer le site via votre serveur local
